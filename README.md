# Mulevents
*A centralized event platform for Colby Residential Life*

**Author:** Ninh Giang Nguyen  
**Project Type:** Solo CS Software Engineering Project (Volunteer)

---

## Overview
**Mulevents** is a full-stack web app designed to help **Colby residents and Community Advisors (CAs)** discover, manage, and promote campus events in one centralized platform.

Built as a solo coursework project, Mulevents focuses on realistic workflows, role-based access, and simple event analytics.

---

## Key Features

### Residents (No Login)
- Browse all CA-hosted events in one feed  
- Add events to **Google Calendar**  
- Refer friends & message hosting CAs  
- **GOAT Events**: Top 3 events of the month by attendance  

### Community Advisors (Login Required)
- Create, edit, promote, or remove events  
- Auto-send promotional emails  
- **QR-code attendance tracking** (replaces p-cards)  
- View attendance-based performance
